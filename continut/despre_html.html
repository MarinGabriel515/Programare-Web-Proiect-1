Limbajul html

HTML(Hypertext Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html se poate gasi la adresahttps://www.w3schools.com/html/html_links.asp

Tag-urile html sunt cele mai utilizate in realizarea tabelelor sunt:

table defineste un tabel
tr defineste un rand
td o celula in un tabel

Observatie: pentru a pune un contur se poate (dar nu se recomanda)sa se puna border="1".

La validare va da eroarea : The border atribute on the table is obsolete

Formularele se pot crea cu ajutorul tag-ului form care poate contine tag-urile: input,textarea,label,select,option,button.

Pentru mai multe detalii consultati cursul sau tutorialulHTML forms and inputs la adresa https://www.w3schools.com/html/html_forms.asp.
